# Stochastic Maximum Coverage with ML4CO

This project implements a minimal ML4CO-style pipeline for stochastic maximum
coverage on graph datasets.

Given:

```python
graph_dataset: list[networkx.Graph]
```

each graph is sampled into `m` stochastic subgraphs. The target for the original
graph is the average objective value of all sampled subgraphs:

```text
y(G) = mean_j objective(H_j)
```

The default objective is graph maximum coverage: select at most `k` seed nodes
and count how many nodes are covered within `cover_radius` hops in the sampled
subgraph. Labels are generated by a greedy approximation by default.

## Project layout

```text
smc_ml4co/
  data.py          sampling, saved scenarios, feature extraction, collate
  preprocess.py    standalone dataset preprocessing (sampling + offline save)
  objective.py     maximum coverage objective and greedy solver
  model.py         lightweight GNN models
  train.py         supervised regression baseline
  unsupervised.py  unsupervised policy training
examples/
  run_demo.py                supervised synthetic demo
  run_unsupervised_demo.py   unsupervised synthetic demo
tests/
  test_pipeline.py
```

## Run unsupervised demo

This is the main ML4CO route for stochastic maximum coverage. It saves all
sampled subgraphs to `scenario_cache/` via standalone preprocessing, then trains
a node policy directly from the expected coverage objective.

First preprocess:

```bash
python -m smc_ml4co.preprocess --scenario-dir scenario_cache --num-graphs 100 --m-samples 16 --k 5 --without-labels
```

Then train with offline files:

```bash
python -m smc_ml4co.unsupervised --load-saved-scenarios --scenario-dir scenario_cache --epochs 30
```

One-command demo (synthetic data generation + online sampling during training):

```bash
python -m examples.run_unsupervised_demo
```

Equivalent CLI (online sampling, no offline save by default):

```bash
python -m smc_ml4co.unsupervised --num-graphs 100 --m-samples 16 --k 5 --epochs 30 --scenario-dir scenario_cache
```

To save sampled subgraphs during training (optional):

```bash
python -m smc_ml4co.unsupervised --num-graphs 100 --m-samples 16 --scenario-dir scenario_cache --save-scenarios-during-train
```

To reuse already-saved sampled subgraphs:

```bash
python -m smc_ml4co.unsupervised --load-saved-scenarios --scenario-dir scenario_cache
```

## Use your own `graph_dataset`

```python
from smc_ml4co.data import StochasticMaxCoverDataset

dataset = StochasticMaxCoverDataset(
    graph_dataset=my_networkx_graphs,
    m_samples=16,
    k=5,
    cover_radius=1,
    node_keep_prob=0.8,
    edge_keep_prob=0.9,
    seed=42,
    compute_labels=False,
    scenario_dir="scenario_cache",
    save_generated=True,
)
```

Each dataset item is a `GraphScenarioSample`:

```python
sample.graph_id
sample.original_graph
sample.subgraphs
sample.subgraph_values
sample.target
```

During training, the collate function flattens all subgraphs in a mini-batch,
runs the GNN over the flattened batch, then pools predictions by original graph.

For unsupervised training, no target label is required. The model outputs node
selection probabilities on the original graph and maximizes:

```text
mean over sampled subgraphs: coverage(policy(G), H_j)
```

with a differentiable budget penalty for the `k` selected nodes.

## Saved scenario format

`scenario_cache/manifest.json` lists one pickle file per original graph. Each file
stores:

```python
graph_id
original_graph
subgraphs          # all m sampled subgraphs
subgraph_values    # zeros for unsupervised mode, labels for supervised mode
target             # zero for unsupervised mode, mean label for supervised mode
```

## Training API for an existing graph_dataset

```python
from argparse import Namespace
from smc_ml4co.unsupervised import train_unsupervised_on_graph_dataset

args = Namespace(
    m_samples=16,
    k=5,
    cover_radius=1,
    node_keep_prob=0.8,
    edge_keep_prob=0.9,
    hidden_dim=64,
    num_layers=3,
    dropout=0.1,
    batch_size=8,
    epochs=30,
    lr=1e-3,
    weight_decay=1e-4,
    grad_clip=1.0,
    temperature=1.0,
    budget_penalty_weight=2.0,
    entropy_weight=0.0,
    seed=42,
    device="cpu",
    checkpoint="",
    scenario_dir="scenario_cache",
    load_saved_scenarios=False,
    no_progress=False,
)

result = train_unsupervised_on_graph_dataset(graph_dataset, args)
```
